Matrix / isolated design system lab

Editorial control room, not generic AI SaaS.

This prototype defines a UI system for Matrix before we wire a single app route. The direction is premium, tactile, and operational: warm paper inserts inside a dark shell, serif identity moments, crisp controls, and domain modules built around auth, chat, boards, and schedules.

Locked starting points

These are the decisions this prototype is built around.

Dark shell + paper inserts The shell stays calm and operational while moments of intent break into warm, editorial surfaces.
Serif for identity only Serif is reserved for headlines and artifact moments so it feels valuable rather than decorative.
Signals, not candy Ember, cobalt, and citron accents behave like meaningful control states.
Chat is the product The component language prioritizes messages, artifacts, composer, and preview first.
